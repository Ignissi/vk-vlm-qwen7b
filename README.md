# VK VLM Practice — Qwen2.5-VL-7B + LoRA

Проект адаптирует `Qwen/Qwen2.5-VL-7B-Instruct` к русскоязычным вопросам по изображениям на открытом датасете `deepvk/MMBench-ru`.

## Что получается

После запуска создаются:

- `adapter/adapter_model.safetensors` — обученные LoRA-веса;
- `results/metrics.json` — accuracy до и после обучения;
- `results/predictions.csv` — ответы модели на контрольных примерах;
- `REPORT.md` — автоматически заполненный подробный отчёт.

## Запуск на Ubuntu 24.04 + RTX 5090

```bash
cd ~/vk-vlm-qwen7b
./setup.sh
./run.sh
```

`setup.sh` создаёт отдельную Python 3.12 venv и устанавливает PyTorch с CUDA 12.8 для RTX 5090.

`run.sh` запускает весь эксперимент: загрузку данных, оценку исходной модели, LoRA-обучение, повторную оценку и сохранение результатов.

## Используемые параметры

- train: 512 примеров;
- internal eval: 128 непересекающихся примеров;
- BF16 LoRA;
- batch size: 1;
- gradient accumulation: 4;
- 2 эпохи;
- learning rate: 5e-5;
- LoRA: `q_proj`, `k_proj`, `v_proj`, `o_proj`.

## Что загружать на GitHub

Загружается вся папка проекта после завершения запуска, кроме `.venv/` и `checkpoints/`. Базовые 7B-веса в репозиторий не копируются.

## Важная оговорка

`MMBench-ru` имеет только split `dev`. Для учебного эксперимента он делится на непересекающиеся обучающую и контрольную части. Полученная accuracy является внутренней метрикой, а не официальным результатом полного MMBench.
